Timeline of programming languages history

Year Events
1843 Ada Lovelace invents the first-ever machine algorithm which lay the foundation for all programming language
1945 Konrad Zuse developed the first ‘real’ programming language called Plankalkül
1949 Assembly language, a type of low-level programming language that simplified the language of machine code was used in the Electronic Delay Storage Automatic Calculator
1949 Shortcode, the first ever high level language, was suggested by John McCauley in 1949. Later, William Schmitt implemented it for the BINAC computer the same year and for the UNIVAC in 1950.
1957 FORmula TRANslation or FORTRAN was created by John Backus and is considered to be the oldest programming language in use today.
1958 Algorithmic language or ALGOL was created, it served as the starting point for the development of some of the most important programming languages.
1959 Common Business Oriented Language (COBOL) was created, which is still used to this day primaly for banking.
1964 Beginners All-Purpose Symbolic Instruction Code or BASIC was developed by a group of students at Dartmouth College
1970 Pascal was developed by Niklaus Wirth.
1972 C was developed by Dennis Ritchie.
1972 SQL was developed by researchers Raymond Boyce and Donald Chamberlain.
1983 Bjarne Stroustrup modified the C language at the Bell Labs and so C++ was created.
1991 Python was developed by Guido Van Rossum.
1995 java was developed by James Gosling. PHP was developed by Rasmus Lerdorf in the same year.
1995 java was developed by James Gosling. PHP was developed by Rasmus Lerdorf. JavaScript was developed by Brendan Eich.
2000 C# was developed at Microsoft.
